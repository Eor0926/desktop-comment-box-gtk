# Changelog

## v1.4.6

- Same-box icon movement uses a floating drag preview and commits to the nearest grid position on release.
- Appearance configuration is isolated per box.
- Removed the standalone Configure Defaults menu item.
- Moved "Use this appearance as defaults for new boxes" into Configure Appearance.
- Keeps selection capture behavior, grid placement, box-to-box file movement, and desktop export behavior from the v1.4.x line.

## v1.4.x line

- Added Nemo/Desktop action: Create Comment Box from Selection.
- Capture only accepts selected files directly inside `~/Desktop`.
- Pressing the capture shortcut with nothing selected does nothing.
- Added grid-based icon placement.
- Added box-to-box file movement and desktop export.

## v1.0.x line

- Initial GTK version replacing the Cinnamon desklet prototype.
- Real file-manager drag/drop support.
- Borderless desktop-style boxes with settings, opacity, title rename, and autostart.
